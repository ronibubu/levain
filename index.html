르방 리프레시 계산기

르방 양을 입력하면 통밀, 강력분, 물 양을 자동으로 계산해요.

g
통밀 10g
강력분 40g
물 40g
최종 르방 총량 140g

르방 50g에 통밀 10g, 강력분 40g, 물 40g을 넣어주세요.

기준 비율 설명

  • 르방 50g + 통밀 10g + 강력분 40g + 물 40g
  • 르방 : 밀가루 : 물 = 1 : 1 : 0.8
  • 밀가루는 통밀 20%, 강력분 80%로 계산해요.

르방 만들기 과정

  1. 1일차: 통밀 50g + 물 50g
  2. 2일차: 1일차 르방 50g + 통밀 50g + 물 50g
  3. 3일차: 2일차 르방 50g + 통밀 50g + 물 50g
  4. 4일차: 3일차 르방 50g + 통밀 50g + 물 50g
  5. 5일차: 4일차 르방 50g + 통밀 10g + 강력분 40g + 물 40g

주의사항

  • 르방이 담긴 통 무게만 재서 통에 적어두면 계량하기 편해요.
  • 실온, 온도, 밀가루 상태에 따라 발효 속도는 달라질 수 있어요.
  • 냄새, 부피 증가, 기포를 함께 확인해주세요.
